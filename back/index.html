Question-2: What is the difference between innerHTML, innerText, and textContent ?
1. innerHTML: Returns or sets the HTML content inside an element, including tags.
2. innerText: Returns or sets the visible text content, ignoring hidden elements and respecting CSS styling.
3. textContent: Returns or sets all text content, including hidden elements, without interpreting HTML tags.
Question-3: What is event delegation in the DOM?
uses a single event listener on a parent element to handle events for its children, leveraging event bubbling. This is efficient for dynamic content and reduces the need for multiple listeners.
Question-4: What is event bubbling in the DOM?
is a behavior in the DOM where an event triggered on a child element propagates (bubbles) up through its parent elements in the hierarchy. For example, if a button inside a div is clicked, the click event bubbles up to the div, then to its parent, and so on, until it reaches the document object. This allows handling events at a higher level in the DOM tree.
Question-5: How do you create, add, and remove elements using JavaScript?
1. Create: Use document.createElement('tagName').
2. Add: Use parentElement.appendChild(newElement) or parentElement.insertBefore(newElement, referenceElement).
3. Remove: Use parentElement.removeChild(childElement) or childElement.remove().
